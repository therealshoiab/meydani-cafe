# Meydani Cafe - Restaurant Website

Welcome to the Meydani Cafe restaurant website project. This is a modern, responsive web application built with React, Vite, and Tailwind CSS v4.

## Technology Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS v4
- **Routing**: React Router DOM v7
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## Folder Structure
- `src/components/`: Reusable UI components (Navbar, Footer, Button, etc.)
- `src/pages/`: Main page views (Home, Menu, About, Contact, Booking)
- `src/assets/`: Images, logos, and local media assets
- `src/data/`: Static data (menu items, testimonials, team, etc.)
- `public/`: Public assets
- `production_artifacts/`: Final production build outputs and assets
- `.agents/`: Agent workflows and custom skills

## Getting Started
To start the local development server:
```bash
npm run dev
```

To build for production:
```bash
npm run build
```
