# Workflow: Build Restaurant Site

This workflow documents the steps to build the Meydani Cafe restaurant website.

## Phase 1: Core Layout & Navigation
- [ ] Design System & Global Styles configuration in `src/index.css`
- [ ] Navbar component with responsive mobile drawer
- [ ] Footer component with location, contact info, and opening hours
- [ ] Router configuration in `src/App.jsx` for all pages

## Phase 2: Page Development
- [ ] Home Page: Hero, featured dishes, about teaser, testimonials
- [ ] Menu Page: Categories, interactive filter, dietary tags, dish details modal
- [ ] About Page: Story, values, chef introduction, and gallery
- [ ] Contact & Booking Page: Address details, interactive map placeholder, reservation form

## Phase 3: Features & Polishing
- [ ] Interactive reservation form with real-time feedback
- [ ] Rich framer-motion page transitions and scroll animations
- [ ] Persistent cart or wishlist for online ordering (if requested)
- [ ] Dark/Light mode toggle
